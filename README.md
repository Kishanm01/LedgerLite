# Ledger Lite Repo

How to run: 
- make sure yarn is installed
- have a .env.local file with these properties:
    1. NEXT_PUBLIC_SUPABASE_ANON_KEY
    2. SERVICE_ROLE_KEY
    3. NEXT_PUBLIC_SUPABASE_URL=https://studio.ledgerlite.us
    4. SENDGRID_API_KEY
- RUN: `yarn`
- RUN: `yarn dev`

Should be up.
